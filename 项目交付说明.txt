===============================================
快速开始
===============================================

1. 安装依赖
   cd rt-voicechat-cli
   pip install -r requirements.txt

2. 配置API密钥
   copy .env.example .env
   编辑.env文件，填入: DEEPSEEK_API_KEY=sk-xxxxx

3. 运行程序
   python -m src.main --text-mode

4. 开始对话
   你: 你好
   AI: (AI回复)
   
   输入 /quit 退出

===============================================
常用命令
===============================================

# 文本模式
python -m src.main --text-mode

# 指定模型

# 指定配置

# Windows快捷方式
run.bat

===============================================
配置示例
===============================================

.env 文件:
DEEPSEEK_API_KEY=sk-your-key
OPENAI_API_KEY=sk-your-key
LOG_LEVEL=INFO

default_config.yaml:
models:
  active: "deepseek"
  deepseek:
    api_key: ""
    model: "deepseek-chat"
    temperature: 0.7
